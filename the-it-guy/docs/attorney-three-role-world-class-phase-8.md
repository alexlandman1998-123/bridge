# Attorney three-role world-class programme — Phase 8

Phase 8 turns per-matter controls into firm-level rollout intelligence for authorised operations leaders.

## Delivered

- Unique-matter portfolio metrics for risk, inactivity, overdue next actions, and required-role coverage.
- Coverage reporting for transfer, bond, and cancellation attorneys.
- Threshold-driven `go`, `observe`, `hold`, or `insufficient_data` decisions.
- A prioritised intervention queue with direct matter links and explicit reasons.
- Firm-wide rollout intelligence is shown only where the existing operations service grants firm-dashboard visibility.

## Exit evidence

Run `npm run test:attorney-three-role-phase8`. It executes the complete Phase 0–7 regression chain, portfolio decision tests, operations wiring checks, and existing workflow command verification.
