# Attorney three-role world-class programme — Phase 5

Phase 5 gives the cancellation attorney a time-bound, dependency-aware command centre on the shared matter workspace.

## Delivered

- Six operational domains: existing bond/bank instruction, cancellation figures and expiry risk, guarantees, drafting/signing, linked lodgement/registration, and settlement close-out.
- Existing cancellation workflow commands power the primary next action.
- Readiness metrics cover missing matter data, documents, signatures, and legal handoffs.
- Transfer and guarantee dependencies are read-only source signals. Cancellation users can request or escalate a handoff from their own lane without mutating transfer or bond work.
- Cancellation-lane permissions control whether actions are executable or coordination-only.

## Exit evidence

Run `npm run test:attorney-three-role-phase5`. It executes the Phase 0–4 regression chain, cancellation cockpit model tests, UI wiring checks, and existing workflow command verification.
